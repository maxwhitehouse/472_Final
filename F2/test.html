Maui Wildfire Recovery as a Spatial Externality Game

This interactive map visualizes a simple game-theoretic model of wildfire recovery and compliance in three Maui locations: Lahaina, Waiohuli, and Kahului.

Each 3×3 grid represents a small neighborhood. Clicking a grid opens sliders for that location. The model evaluates whether compliance is the dominant strategy using the condition:

Lθ + pF > c

Where:

  • L = expected wildfire loss
  • θ = mitigation effectiveness
  • p = enforcement probability
  • F = penalty for non-compliance
  • c = cost of compliance

How to use this map

  1. Click one of the three grids.
  2. Adjust the sliders for that location.
  3. Watch the grid update color based on the equilibrium.
  4. Use the basemap switcher or zoom buttons to explore the map.
